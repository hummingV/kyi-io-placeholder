Just a placeholder static site based on [Bootstrap - Bare](https://startbootstrap.com/template-overviews/bare/)

### Run
~~~
npm run start
~~~

### build
~~~
npm run build
~~~

### publish to S3
~~~
./publish.sh
~~~
